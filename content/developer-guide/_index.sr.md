Овај водич је намењен програмерима који желе да разумеју унутрашњост LibreSCRS-а, изграде пројекат из изворног кода, прошире систем новим додацима за картице, или користе C++23 SDK из друге апликације.

## Теме

- [Преглед архитектуре]({{< ref "developer-guide/architecture" >}}) — компоненте система, распоред јавног API-ја, модел додатака, конвенције власништва + руковања грешкама, ток података од убацивања картице до приказа.
- [Архитектура потписивања]({{< ref "developer-guide/signing-architecture" >}}) — нативни механизам потписивања, подржани формати, руковање сертификатима, end-to-end ток потписивања.
- [Водич за интеграцију потписивања]({{< ref "developer-guide/signing-integration" >}}) — интеграција дигиталног потписивања у апликације користећи LibreMiddleware.
- [SDK референца]({{< ref "developer-guide/sdk-reference" >}}) — наративна референца по простору имена за `LibreSCRS::{Auth,SmartCard,Plugin,Signing,Secure}`, walkthrough за писање додатка, миграциони напомене са 3.x.
- [API политика]({{< ref "developer-guide/sdk-reference/api-policy" >}}) — верзионисање (SemVer), ритам застаревања, валидација vs. runtime руковање грешкама, правила Plugin ABI-ja.
- [Изградња из изворног кода]({{< ref "developer-guide/building-from-source" >}}) — предуслови, команде за изградњу, покретање тестова, sdk-consumer пример.
